### 🎨 如何更改预览窗口内容



您的所有“内容”都定义在 `GeophysicalPreviewControl.cs` (用户控件) 文件中。您有两种方式来修改它：



#### 1. 方式一：使用“可视化设计器” (添加控件)



1. 在 Visual Studio 的“解决方案资源管理器”中，**双击 `GeophysicalPreviewControl.cs`**。
2. 这会打开一个**可视化的空白画板**。
3. 从 Visual Studio 左侧的“**工具箱**”中，拖拽您需要的任何控件到这个画板上。
   - **例如**：拖入一个 `DevExpress.XtraGrid.GridControl` (用于显示表格) 或 `DevExpress.XtraCharts.ChartControl` (用于显示图表)。
4. 在右下角的“属性”窗口中，修改您拖入控件的 `(Name)` 属性，比如命名为 `myGridControl` 或 `myChartControl`。



#### 2. 方式二：修改代码 (绑定数据)



这是最关键的一步。您需要告诉控件“要显示什么数据”。

1. 打开 `GeophysicalPreviewControl.cs` 的**代码视图**（右键点击文件 -> "查看代码"）。
2. 找到 `public void Initialize(MapLayer layer)` 方法。
3. **在这个方法内部，编写您的数据处理逻辑。**

### 💡 示例：显示栅格图层的属性



假设您的地球物理数据是栅格图层（`RasterLayer`），您想在预览窗口中显示它的统计信息（最大值、最小值等）。

**步骤：**

1. **(设计器)** 拖拽一个 `DevExpress.XtraGrid.GridControl` 到您的 `GeophysicalPreviewControl` 画板上，并将其 `(Name)` 设置为 `gridControl1`。
2. **(代码)** 修改 `GeophysicalPreviewControl.cs` 的代码如下：

**`GeophysicalPreviewControl.cs` (示例代码)**

C#

```
using System.Windows.Forms;
using MapGIS.GeoMap;
using MapGIS.GeoDataBase; // (重要) 添加引用，为了 RasterDataSet
using MapGIS.GeoDataBase.GeoRaster; // (重要) 添加引用，为了 RasterLayer
using System.Data; // (重要) 添加引用，为了 DataTable

namespace MapGISPlugin3
{
    public partial class GeophysicalPreviewControl : UserControl
    {
        // ... (构造函数 InitializeComponent() 不变) ...
        // public GeophysicalPreviewControl() { ... }

        /// <summary>
        /// (核心) 在这里填充您的控件
        /// </summary>
        public void Initialize(MapLayer layer)
        {
            if (layer == null)
            {
                // (可选) 清空表格
                // gridControl1.DataSource = null; 
                return;
            }

            // 1. 检查图层是否为栅格图层
            if (layer is RasterLayer)
            {
                // 2. 将其转换为 RasterLayer
                RasterLayer rasLayer = layer as RasterLayer;

                // 3. 获取其数据源 (RasterDataSet)
                RasterDataSet rasDataset = rasLayer.GetData() as RasterDataSet;
                
                if (rasDataset != null)
                {
                    // 4. 创建一个 DataTable 来显示数据
                    DataTable dt = new DataTable();
                    dt.Columns.Add("属性", typeof(string));
                    dt.Columns.Add("值", typeof(string));

                    // 5. 填充数据
                    dt.Rows.Add("图层名称", rasLayer.Name);
                    dt.Rows.Add("图层类型", rasLayer.Type);
                    dt.Rows.Add("宽度 (像元)", rasDataset.Width);
                    dt.Rows.Add("高度 (像元)", rasDataset.Height);
                    dt.Rows.Add("波段数", rasDataset.BandCount);

                    // (获取第一个波段的统计信息)
                    if (rasDataset.BandCount > 0)
                    {
                        RasterBand band = rasDataset.GetRasterBand(1); // 获取第一个波段
                        dt.Rows.Add("波段1: 最大值", band.MaxValue);
                        dt.Rows.Add("波段1: 最小值", band.MinValue);
                        dt.Rows.Add("波段1: 平均值", band.MeanValue);
                    }

                    // 6. (关键) 将 DataTable 绑定到您在设计器中添加的 GridControl
                    gridControl1.DataSource = dt;
                }
            }
            else if (layer is VectorLayer)
            {
                // 如果是矢量图层，您也可以做类似处理
                // VectorLayer vecLayer = layer as VectorLayer;
                // ...
            }
        }
    }
}
```

以后您想做任何修改，流程都是：

2. 打开 `GeophysicalPreviewControl.cs` 的**代码**，修改 `Initialize(MapLayer layer)` 方法，将 `layer` 中的数据绑定到您的UI控件上。
